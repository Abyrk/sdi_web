About this page

This page gives more information about how the web pages have been organised and checked.

Checking the Website

Browser compatibility

The website works best in Google Chrome or Firefox. Much of the content works in Internet Explorer, but some of the maps (e.g. Google Earth) is more temperamental. It is also best viewed with the windows maximised.

CSS validation

The main CSS file "moderna.css" used to style these pages was checked successfully using the CSS validator.

HTML validation

The main pages of this website have been checked successfully using the HTML validator as meeting the XHTML 1.0 Transitional. The only exception is the "obsorne_house.htm" file which has 1 error that seems to be to do with the use of the map tag. I have not checked the subfolders created as a result of downloading ESRI templates and Web Apps.

Website Structure

The index page is at the top of the folder structure (i.e. sswf7303f/acb_iow/index.htm). All other files are in folders below that. There are separate folders for the CSS, data, images, pages, and scripts that I have written. For the more complex ESRI web app templates and downloads I kept all the information together in a folder for each app within the 'pages' folder.
