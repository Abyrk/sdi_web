About this page

This map uses a leaflet map. I've written a JavaScript file with a jQuery to read an XML file of points and add them to the map. The script puts all the markers with the same rating into an array. The arrays are then used in functions that are called when the user clicks the appropriate button (below the map) to show or hide the markers for establishments with each rating.

Where to eat on the Isle

This map shows the food hygiene rating for cafes, restaurants, pubs, hotels and bed and breakfasts around the Isle as at 25 Feb 2015 according to the Food Standards Agency rating system. Use the buttons below the map to show/hide establishments with different ratings. Click on a flag to see the establishment's name.

Ratings: Very good Good Generally satisfactory Improvement necessary Major or urgent improvement necessary

       

Source: Food Standards Agency
