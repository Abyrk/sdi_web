About this page

Population

The population density map uses data from the 2011 population census at the lower layer super output area of geography. This was obtained from the ONS neighbourhood statistics portal.

I used ArcMap to link the population numbers and geography and then used the toolbox to create a kmz file of the map. After saving the KMZ file to the public server I imported it to Google Earth Pro (which is now free). That meant I could use the apply style template function to the colour and height of each region according to the population density.

Average house prices

I used ArcMap to define and style the house price information. I then saved it to the ArcGIS server, shared it and used it in the online Web App Builder to create the final output. Once that was done and shared I imported it to the local version of Web App Builder and downloaded it for use on this web page.

Isle of Wight facts and figures

Population

The average density of the usual resident population of the Isle of Wight is 3.6 persons per hectare (ONS: Neighbourhood Statistics, 2011). At that time the median age of the population was 46 years of age. The map below shows how the population density varies across the Isle.
