About this page

The first map shown here is an ESRI 'accordion' style story map. To create this map I followed the instructions available from the ArcGIS website. Each map was produced in ArcGIS online and shared publicly. There seems to be a problem getting the map to display correctly in a frame in Internet Explorer (though it is fine in full screen)

The second map uses OpenLayers, NSRI natural perils layers on GeoServer and some standard base layers. In GeoSever new styles were added so the same data could be re-used to show different information (flood risk, wind exposure and silt-related erosion) I found code to add the legends from GeoServer and these are added as images below the map. A further refinement would be to switch these on and off as the user chooses different layers.
